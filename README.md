# Tourism Employment and GDP Contribution

#### Mini Project ‚Äî PSY6422 Data Analysis & Visualisation

This project explores the relationship between **tourism employment** and **tourism-related contribution to GDP** across multiple countries over time. The analysis aims to understand how economic dependence on tourism aligns with labour patterns in tourism-related industries.

The datasets used in this project originate from **Our World in Data** (<https://ourworldindata.org/>), containing yearly estimates of: - Number of individuals employed in tourism-related industries per 1000 people\
- Tourism GDP as a proportion of total GDP

Both datasets are publicly accessible and provide consistent, comparable measures across countries.

## üìä Project Overview

This project investigates:

1.  **How tourism employment changes across different countries over time**\
2.  **Whether higher GDP reliance on tourism predicts higher employment in tourism-related industries**\
3.  **Which regions show the strongest tourism dependence**\
4.  **Visual comparisons of employment trends and GDP contribution trends**

The project includes: - Data cleaning and tidying\
- Exploratory visualisations using ggplot2\
- Correlation analysis\
